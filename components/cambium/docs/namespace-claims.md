# Namespace claims

The following crates.io names were claimed by this project on 2026-07-13:

- `cambium`
- `meristem`
- `sprigging`
- `genet`
- `genet-stylo`

`cambium-nematic` was claimed and released from this workspace on 2026-07-14.

Cambium, Meristem, Sprigging, and Cambium Nematic belong to this workspace.
`genet` names the adjacent browser engine, and `genet-stylo` names its Stylo
fork package. A claimed name establishes namespace ownership; it does not by
itself mark an implementation release as ready.
